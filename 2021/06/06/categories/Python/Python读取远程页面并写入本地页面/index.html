Python例子-读取远程页面并写入本地页面

Python 2021/06/06

import urllib.request
response = urllib.request.urlopen(‘http://www.baidu.com')
‘’’ 方法1

读取文件的全部内容，把读取到的数据赋值给一个字符串变量

data = response.read()

print(type(data)) # <class ‘bytes’>

print(data)

with open(‘baidu.html’,’wb’) as f:
f.write(data)
